Hidden Script inside CSS#

There is a script hidden inside the two CSS files. These two .css files together can detect if the computer is connected to the internet. If not they block the viewing of the data inside

The main files are:-

  1. conversation-small-gncbdz17
  2. root-c2ck6e7e

It starts with conversation-small-gncbd17 This makes a call to https://cdn.oaistatic.com/assets/manifest-8565d6fb.js https://cdn.oaistatic.com/assets/hwhtsahnjjraw24h.js

Then it makes a call to root-c2ck6e7e. this can be happening anywhere between the html page and the two css files. The webpage then makes a request to prefetchCSSResource.js which is really its own thing. No source file. NoScript blocks script-src-element so JS doesn’t execute in Firefox
